c00lploit v1.0.1 Documentation

Overview

c00lploit v1.0.1 is a modular Lua-based UI library for Roblox, providing draggable windows, tabs, buttons, toggles, sliders, dropdowns, and notifications.

Features

  • Draggable main window
  • Side tab navigation
  • Scrollable sections
  • UI Components: Buttons, Toggles, Sliders, Dropdowns
  • Notifications stacked in top-right corner
  • Version label display

Installation

Insert the Lua script into a LocalScript in Roblox (e.g., inside StarterPlayerScripts).

Usage

local c00lploit = require(game.ServerScriptService.c00lploit)

local objGen = c00lploit.getObjGen()
local Gui = objGen.getObjects()

local tab1, tabBtn1 = objGen:CreateTab("Tab 1", Gui)
local section1 = objGen:CreateSection("Section 1", tab1)

objGen:CreateButton("Click Me", section1, function()
    objGen:Notify(Gui, "Button Clicked!")
end)

API Reference

getObjGen()

Returns an object generator with methods to create UI components.

getObjects()

Initializes and returns the GUI components.

CreateTab(name, parent)

Creates a new tab with the specified name under the given parent.

  • name: string — tab name
  • parent: GUI object — parent window

Returns the created tab folder and the tab button.

CreateSection(name, parent)

Creates a scrollable section inside a tab.

CreateButton(name, parent, callback)

Creates a button.

objGen:CreateButton("Button Name", parentSection, function()
    print("Clicked!")
end)

CreateToggle(name, parent, callback)

Creates an on/off toggle.

CreateSlider(name, parent, min, max, callback)

Creates a slider.

CreateDropdown(name, parent, options, callback)

Creates a dropdown menu with selectable options.

Notify(parent, text)

Displays a notification at the top-right corner of the screen.

Example: loadstring Usage

You can execute external scripts using loadstring:

loadstring(game:HttpGet("https://rawscripts.net/raw/Universal-Script-bang-gui-36038", true))()

Customization

Gui.MainWindow.BackgroundColor3 = Color3.fromRGB(50,50,50)

Create custom buttons:

local customButton = Instance.new("TextButton")
customButton.Size = UDim2.new(0,200,0,50)
customButton.Text = "Custom Button"
customButton.BackgroundColor3 = Color3.fromRGB(100,100,100)
customButton.Parent = section1
